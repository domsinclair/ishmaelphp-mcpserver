# Contributing to Ishmael MCP Server

Thank you for your interest in contributing to the Ishmael MCP Server!

## How to Contribute

1. **Fork the Repository**: Create your own fork of the project on GitHub.
2. **Clone the Repo**: `git clone https://github.com/domsinclair/ishmaelphp-mcpserver.git`
3. **Create a Branch**: Create a feature branch for your changes.
4. **Install Dependencies**: Run `composer install`.
5. **Make Changes**: Implement your feature or fix.
6. **Run Tests**: Ensure all tests pass by running `composer test`.
7. **Submit a Pull Request**: Push your branch to your fork and open a PR against the main repository.

## Development Standards

- Follow PSR-12 coding standards.
- Ensure new features include corresponding unit tests.
- Update documentation if you are changing functionality.
